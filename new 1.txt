# 2. Railway dashboard
# - New Project → Deploy from GitHub repo

Cách lấy Railway URL (chuẩn nhất)

Vào Railway.app → đăng nhập

New Project → Deploy from GitHub Repo

Chọn repo: dinh-hub/xiaozhi-music-backend

Railway sẽ build + chạy

Sau khi chạy xong:

Vào project → tab Settings hoặc Domains / Networking

Bấm Generate Domain (hoặc “Add Domain”)
